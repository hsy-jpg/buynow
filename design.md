# 물가 체감 앱 디자인 시스템

> 참고 레퍼런스: lime (심리상담 앱) — 연두빛 + 둥근 일러스트 + 친근한 캐릭터 톤
> 이 문서는 Cursor/Claude Code에 그대로 전달해서 컴포넌트 스타일 가이드로 쓸 수 있도록 작성했습니다.

---

## 1. 브랜드 톤 & 방향

- **한 줄 컨셉**: "예측하지 않고, 사실만 정직하게 보여주는 물가 친구"
- **톤앤매너**: 딱딱한 통계 앱이 아니라, 옆에서 "이번 달은 이래" 하고 알려주는 친근한 친구 같은 느낌
- **레퍼런스에서 가져올 것**: 연두 계열 메인 컬러, 부드러운 블롭(blob) 캐릭터, 라운드 카드, 파스텔 배지
- **레퍼런스와 달라져야 할 것**: 상담 앱의 "위로/공감" 톤 대신, 물가 앱은 "명확함/신뢰"가 조금 더 앞서야 함 — 일러스트는 귀엽되, 정보(그래프, 신호등, 숫자)는 또렷하고 안 흐트러지게

---

## 2. 컬러 팔레트

| 이름 | 역할 | 값 |
|---|---|---|
| `--color-lime-primary` | 메인 브랜드 컬러, 주요 버튼/헤더 | `#7ED321` |
| `--color-lime-soft` | 배경 그라데이션, 카드 배경 강조 | `#E8F7D9` |
| `--color-mint-bg` | 전반적 배경(화이트에 가까운 민트) | `#F4FAF0` |
| `--color-ink` | 본문 텍스트 | `#1F2A1A` |
| `--color-ink-soft` | 보조 텍스트, 설명 | `#6B7568` |
| `--color-card-white` | 카드 배경 | `#FFFFFF` |

**신호등 3색 (구매 타이밍용 — 브랜드 연두와 겹치지 않게 별도 관리)**

| 상태 | 값 | 용도 |
|---|---|---|
| 🟢 사기 좋음 | `#4CAF50` | 신호등 초록 (브랜드 lime보다 살짝 짙게 — 혼동 방지) |
| 🟡 보통 | `#FFC107` | 신호등 노랑 |
| 🔴 비쌈 | `#F1553E` | 신호등 빨강 |

> **디자인 노트**: 브랜드 lime(`#7ED321`)과 신호등 초록(`#4CAF50`)을 같은 색으로 쓰면 "이게 브랜드 색인지 신호인지" 헷갈릴 수 있어서 톤을 분리했어요. 신호등 영역에서만 채도 낮은 초록을 쓰고, 나머지 UI는 밝은 lime을 씁니다.

---

## 3. 타이포그래피

- **디스플레이/제목**: Pretendard Bold — 큰 숫자(변동률 %), 카드 헤드라인
- **본문**: Pretendard Regular/Medium — 설명 텍스트
- **데이터/숫자 강조**: Pretendard SemiBold, tabular-nums 적용 (가격/퍼센트 숫자가 나란히 정렬되게)

| 용도 | 크기 | 굵기 |
|---|---|---|
| 화면 타이틀 | 22px | Bold |
| 카드 헤드라인 | 17px | SemiBold |
| 변동률 큰 숫자 | 28px | Bold |
| 본문/설명 | 14px | Regular |
| 캡션/배지 텍스트 | 12px | Medium |

---

## 4. 레이아웃 & 컴포넌트 톤

- **카드**: radius 20px, 그림자 약하게(`0 2px 12px rgba(0,0,0,0.06)`), 배경 흰색 또는 lime-soft
- **버튼**: pill 형태(radius 999px), 메인 버튼은 lime-primary 배경 + 흰색 텍스트
- **배지(태그)**: radius 999px, 파스텔 배경(연노랑/연초록) + 진한 텍스트 — 레퍼런스의 "친구관계", "상담가능" 배지 톤 그대로 차용
- **하단 네비게이션**: 아이콘 + 라벨, 선택된 탭만 lime-primary로 강조, 나머지는 회색
- **일러스트 캐릭터**: 레퍼런스의 블롭 캐릭터(귤/딸기 등 동글동글한 존재감) 컨셉을 차용해 **"콩(콩나물) 캐릭터"** 또는 **"채소 블롭 캐릭터"**를 마스코트로 제안 — 홈 화면 상단, 빈 상태(empty state), 신호등 결과 화면에 표정이 바뀌며 등장(사기 좋음=웃는 표정, 비쌈=울상)

---

## 5. 화면별 구성

### 5-1. 홈 화면

```
┌─────────────────────────┐
│  [마스코트] 콩나물지수 🔔 👤 │  ← 상단 헤더, 캐릭터 아이콘
├─────────────────────────┤
│  이번 달 TOP5             │
│  📈 많이 오른 상품         │
│  ┌───┐┌───┐┌───┐        │  ← 가로 스크롤 카드
│  │계란││양파││..│         │     (품목 일러스트 + 상승률 %)
│  └───┘└───┘└───┘        │
│  📉 많이 내린 상품         │
│  ┌───┐┌───┐┌───┐        │
├─────────────────────────┤
│  내가 찜한 품목            │
│  ┌─────────────────┐    │
│  │ 계란 🥚          │    │
│  │ 1년 평균보다      │    │
│  │ 4% 비싸요 🔴      │    │  ← 찜한 품목 통계 카드
│  └─────────────────┘    │
├─────────────────────────┤
│  🍠 지금 저렴한 제철 식료품 │
│  ┌─────────────────┐    │
│  │ [배추 일러스트]   │    │
│  │ "배추는 보통      │    │
│  │  9-10월이 가장    │    │  ← 카드뉴스 형태, 스와이프 가능
│  │  저렴해요"        │    │
│  └─────────────────┘    │
├─────────────────────────┤
│ 🏠   📊   👤              │  ← 하단 네비 (홈/메인/마이)
└─────────────────────────┘
```

**포인트**
- TOP5는 가로 스크롤 카드 — 품목 일러스트(귀여운 계란/양파 캐릭터화) + 상승·하락률 크게 표시
- 찜한 품목 카드는 "1년 평균 대비 위치"를 막대/게이지 형태로 시각화하면 좋음(텍스트만보다 직관적)
- 제철 카드뉴스는 레퍼런스의 "친구들의 고민 이야기" 카드 톤을 그대로 차용 — 좌측 상단 작은 태그(예: "제철정보") + 헤드라인 + 일러스트

---

### 5-2. 메인(탐색) 화면

```
┌─────────────────────────┐
│  🔍 품목 검색   [필터 ▾]  │
├─────────────────────────┤
│  [배추] [양파] [계란] ... │  ← 최근/인기 품목 칩
├─────────────────────────┤
│                          │
│      [가격 추이 그래프]    │  ← 라인 그래프 + 평균선 점선
│                          │     (연도별 같은 달 비교 토글 가능)
├─────────────────────────┤
│   🟢 지금 사기 좋아요      │  ← 신호등 결과, 큰 원형 배지
│   "최근 1년 평균보다       │
│    12% 저렴해요"          │
├─────────────────────────┤
│  [❤️ 찜하기]  [🔔 알림설정] │
└─────────────────────────┘
```

**포인트**
- 신호등은 화면에서 가장 눈에 띄는 요소로 — 색이 있는 원/카드 형태로 크게, 근거 문장은 바로 아래 한 줄
- 그래프 위에 "평균선"을 점선으로 표시해서 지금 위치가 평균 위/아래인지 시각적으로 바로 보이게
- 캐릭터가 신호등 옆에서 표정으로 반응(사기 좋음=웃음, 비쌈=울상)하면 레퍼런스의 마스코트 활용 톤과 일관성 있음

---

### 5-3. 마이페이지

```
┌─────────────────────────┐
│  [로그인/회원가입 배너]     │  ← 비로그인 시
├─────────────────────────┤
│  👤 내 계정            > │
│  ❤️ 찜한 품목           > │
│  🔔 알림 설정           > │
│  📒 소비 기록(가계부)    > │
│  🛒 장바구니 시뮬레이터  > │
├─────────────────────────┤
│  [찜한 품목 리스트]        │
│  계란 🥚  +4%  🔴          │
│  양파 🧅  -8%  🟢          │
├─────────────────────────┤
│  [장바구니 시뮬레이터]      │
│  이번 달에 사면: 23,400원  │
│  다음 달 예상: 21,900원    │
│  → "다음 달이 더 유리해요" │
└─────────────────────────┘
```

**포인트**
- 메뉴 리스트는 레퍼런스의 "마이페이지" 화면 톤(아이콘+텍스트+화살표) 그대로 차용
- 장바구니 시뮬레이터는 "이번 달 vs 평소 평균" 비교로, 결과 카드에 결론 한 줄(굵게)을 꼭 넣기
- 소비 기록(가계부)은 품목별 표 형태 — 데이터 밀도가 높으니 이 화면만 카드보다 테이블 컴포넌트 사용

---

## 6. 시그니처 요소 (이 앱만의 기억 포인트)

**"품목 캐릭터화"** — 계란, 양파, 배추 등 주요 식료품을 레퍼런스의 블롭 캐릭터 톤으로 의인화해서, 가격 상태에 따라 표정이 바뀌는 아이콘 세트를 만드는 것을 제안해요. (예: 계란값 오르면 "당황한 계란", 내리면 "신난 계란") 이게 있으면 딱딱한 물가 데이터 앱이 아니라 "우리 앱만의 캐릭터가 있는 서비스"로 차별화돼요. 해커톤 발표에서도 비주얼 임팩트 포인트가 됩니다.
